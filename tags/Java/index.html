    1. ArrayList

    ArrayList特点：

    • ArrayList底层维护一个Object数组(elementData)，新增、查询快，删除、插入慢(慢在会有一次元素复制，快在寻址)，适合顺序添加、随机访问场景；
    • 允许插入null元素，数据允许重复、线程不安全；
    • 初始化时默认容量为10；

    添加元素

    在elementData末尾添加元素。扩容操作：容量变为1.5倍，调用Arrays.copyOf(elementData, newCapacity)将元素复制到新数组里。
